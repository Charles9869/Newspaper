# Newspaper

Application where you can search news articles. Built using the MERN stack

### Version 2

- Authentication
- Comment news article

## TECH STACK

- React, Redux, Styled Components
- NodeJS, Express
- News API

## RUN LOCALLY

```
$ git clone https://github.com/Charles9869/Newspaper
```

```
$ cd server
$ yarn install
$ yarn start
```

```
$ cd client
$ yarn install
$ yarn start
```
